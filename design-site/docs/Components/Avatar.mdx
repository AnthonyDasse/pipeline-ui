# Avatar

A basic avatar component. Use avatars to visually represent entities in your product, for example people, tokens, currency and companies.

import {Avatar} from 'pipeline-ui'

<Avatar
  size="large"
  src="https://media-exp1.licdn.com/dms/image/C4E03AQF82SqrJWUeHA/profile-displayphoto-shrink_200_200/0/1573427081227?e=1631145600&v=beta&t=u0vbwaX3XJrd5wbOOnDLQJ5z0eUIw0B3QRms0v9AiQY"
/>

### Props

Props
alt
string
(optional)
Sets alt attribute on image for accessibility
Default value:

'Avatar'

bg
(optional)
Default value:

'grey'

size
string
(optional)
Sets avatar height and width
Default value:

'2rem'

src
string
(required)
Sets avatar image
theme
object
(optional)
Sets theme

Styled System Props

Avatar uses COMMON, LAYOUT, POSITION, FLEXBOX, BORDERS groups of Styled System props.
