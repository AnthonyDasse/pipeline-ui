# Avatar

A basic avatar component. Use avatars to visually represent entities in your product, for example people, tokens, currency and companies.

import {Avatar} from 'pipeline-ui'

<Avatar
  size="large"

  <tr><td>src</td><td>string (required)</td><td></td><td>Sets avatar image.</td></tr>

    </Table>


<br></br>Avatar also uses COMMON, LAYOUT, POSITION, FLEXBOX, BORDERS groups of Styled System props.
