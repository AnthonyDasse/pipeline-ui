# Button

Use the Button component when you want users to perform an action. It contains a label with an optional icon before or after the label. It maps to a `<button type='button'>` element and extends Box.

import {Button, Flex} from 'pipeline-ui'

<Flex alignItems={"center"}><Button icon="Send" mr={3}>Send</Button><Button icon="Send" iconpos="right" mr={3}>Send</Button><Button icon="MoreHoriz" icononly /></Flex>

## Usage Example

```jsx
<Flex alignItems={"center"}>
  <Button icon="Send" mr={3}>
    Send
  </Button>

  <Button icon="Send" iconpos="right" mr={3}>
    Send
  </Button>

  <Button icon="MoreHoriz" icononly />
</Flex>
```
