# Input

Use Input to build forms for your interface. This component provides a styled version of HTML input.

import {Input} from 'pipeline-ui'

<Input type="text" required={true} placeholder="my note" />

## Usage Example

```jsx
<Input type="text" required={true} placeholder="my note" />
```
