# Link

The Link component provides a basic styled link.

import {Link} from 'pipeline-ui'

<Link href="#!" target="_blank" title="This link goes somewhere">
  Click me!
</Link>

## Usage Example

```jsx
<Link href="#!" target="_blank" title="This link goes somewhere">
  Click me!
</Link>
```
