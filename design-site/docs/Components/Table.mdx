# Table

Table renders the `<table>` HTML element and inherits props from the Box component.

import {Table} from 'pipeline-ui'

<Table>
  <thead>
    <tr>
      <th>Transaction ID</th>
      <th>Value</th>
      <th>Recipient</th>
      <th>Time</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>K4D...FGE</td>
      <td>0.10 Algos</td>
      <td>ER6...JSD</td>
      <td>March 28 2019 08:47:17 AM +UTC</td>
    </tr>
    <tr>
      <td>S3G...230</td>
      <td>0.11 Algos</td>
      <td>4GJ...1E1</td>
      <td>March 28 2019 08:52:17 AM +UTC</td>
    </tr>
    <tr>
      <td>8H6...C40</td>
      <td>0.12 Algos</td>
      <td>6H6...93J</td>
      <td>March 28 2019 08:55:17 AM +UTC</td>
    </tr>
  </tbody>
</Table>
