# Text

The Text component allows you to control font size, weight, alignment and colour.

import {Text} from 'pipeline-ui'

<Text fontFamily="serif">The mission of Algorand is to...</Text>

## Usage Example

```jsx
<Text fontFamily="serif">The mission of Algorand is to...</Text>
```

### Props

fontSize
(optional)
Default value:

2

fontWeight
(optional)
Default value:

2

lineHeight
(optional)
Default value:

'copy'

theme
object
(optional)
Default value:

theme

Styled System Props

Text uses COMMON, TYPOGRAHY groups of Styed System props.
