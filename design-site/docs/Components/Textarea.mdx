# Textarea

The Textarea component provides a styled version of the HTML `<textarea>`.

import {Textarea} from 'pipeline-ui' 

<Textarea placeholder="Start typing..." rows={4} />

## Usage Example

```jsx
<Textarea placeholder="Start typing..." rows={4} />
```

### Props

height
(optional)
Default value:

'auto'

lineHeight
(optional)
Default value:

'copy'

theme
object
(optional)

Styled System Props

Textarea uses COMMON, LAYOUT, POSITION, FLEXBOX, BORDERS groups of Styled System props.
